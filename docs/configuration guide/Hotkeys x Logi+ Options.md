## F1-F4 Keys (Core Functions)
- F1: Open Command Palette (Shift+Control+P)
- F2: Quick Switcher (Shift+Control+O)
- F3: Toggle Reading/Edit View (Shift+Control+E)
- F4: Search in all files (Shift+Control+F)
## F5-F8 Keys (Navigation)
- F5: Toggle left sidebar (⌘+←)
- F6: Toggle right sidebar (⌘+→)
- F7: Open graph view (Shift+Control+G)
- F8: Show File Explorer (Control+Option+F)
## F9-F12 Keys (Content Management)
- F9: Create new note (⌘+Shift+N)
- F10:  Insert template (⌘+Shift+T)
- F11: Create note in new pane (⌘+Control+N)
